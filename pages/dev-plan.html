These are steps to follow.

This list is a paraphrasing of the development strategy outlined in the Front End Web Developer program at Udacity.

  1. Develop an overall structure of the project.
  2. Draw some mockups of the different pages.
  3. Start putting together a design system.
  4. Start building the main page with HTML.
  5. Build the blog post HTML.
  6. Make an HTML page for the design system.
  7. Commence with CSS styling for the pages.
  8. Iterate on initial mockups.
  9. Create links between the pages
  10. Test and iterate on the design and the build.
